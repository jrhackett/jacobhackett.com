About Me

I am a senior at Worcester Polytechnic Institute studying Robotics Engineering and Computer Science. I have recently been working in software and web development. As my double major suggests, my interests are quite broad and include collaborative and interactive robotics, software development, design and more.

My Latest Projects

Interactive Robot

I'm working on my senior capstone project with four colleagues right now. We are developing an interactive segway robot that will operate in busy hospital settings to help patients and loved ones.

Navigation Robot

I worked with two teammates on an autonomous navigation and mapping project. We used a Turtlebot and ROS to explore an unknown environment and map it in real time. I worked primarily on the search algorithm and state control.

Coke or Pepsi

I created an application using Ruby on Rails that allows users to categorize their favorite restaurants by which product line of drinks they serve. I implemented geocoding and tied in Google Maps so you can easily find local restaurants that offer your preferred drink.

Colors

I created a fast paced, mouse accuracy game using Java and JavaFX. It includes a scoring system, sound, and a scaling difficulty based on how long you have played that round.

Artificial Intelligence

I solved Connect-N using adversarial search methods, developed a neural network to predict the output of a set of numbers, and created a decision tree for determining the winner of Connect 4 based on board states containing any number of pieces. I will make the repository for this public soon.

D&D Spells App

I created an Angular application that gives Dungeons and Dragons players a place to search through the spells and see all the details of each spell.

Code Snippets

I made a MEAN.js application that allows users to upload and share code snippets with friends that included syntax highlighting and a simple, easy to use interface.

Fire Fighting Robot

I participated on a team of three to design, build, and program a robot that navigated a complex maze to find and extinguish a candle. I primarily handled the programming and testing of the robot.

Seven Hills Foundation

I worked with three other individuals to develop and implement a plan to introduce smart technology into assisted living homes. We worked closely with Seven Hills Foundation and their assistive technology head. We were able to definitively increase the independence of several people with disabilities through the implementation of this plan.

Manipulation Robot

With two teammates, I developed an embedded 2 link robotic arm. Using forward and inverse kinematics, we were able to draw a triangle and pick up blocks moving on a conveyor belt.

CapraNav

I worked on a team of nine to develop a sophisticated navigation application for WPI's campus. The desktop app can provide directions from anywhere on campus to anywhere else on campus among other features. I designed the entire application, organized our backend and implemented much of our front end.

Cryptocoin Trading

I developed a set of algorithms to trade cryptocurrencies on Poloniex.com. Right now, I can identify and exploit triangular arbitrage opportunities. The script is written in PHP and uses the Poloniex API to access information about the currencies. This project is entirely written and maintained by me.

Personal Website

This website is the result of years worth of redesigns and learning. Not only is a portfolio, but it is also a place for me to practice web design and development. If you have any suggestions for how to improve, please let me know!
